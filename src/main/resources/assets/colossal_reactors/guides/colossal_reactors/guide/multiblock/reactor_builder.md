# Reactor Builder

<BlockImage id="reactor_builder" scale="4" />

The **Reactor Builder** helps you **place** casing, glass, interior components, and ports from supplied materials instead of hand-placing every block.

## Typical workflow

1. **Place** the builder against your reactor footprint (orientation matters).
2. **Open** the screen and load or define the **pattern** (dimensions, layers, allowed blocks).
3. Use **preview** if available so you can see planned positions before committing.
4. **Build** when you are ready; supply the buffer with the correct items.
5. **Mark input** (not a generic “mark”): use it to **assign buffer slots to specific items** so the builder knows what to pull for each step—follow on-screen labels and tooltips.

Exact names depend on your game version; read the button text while operating the builder.
