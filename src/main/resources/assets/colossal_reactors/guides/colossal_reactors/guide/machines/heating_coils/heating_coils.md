# Using heating coils

Each coil variant shares the same general interaction pattern; recipe specifics come from world datapacks and **JEI** (when installed).

## Placement

- Orient the block so the **front face** matches where you want pipes or players to interact; capability sides follow the block’s facing unless the definition allows all sides.

## When you open the screen

Depending on the coil definition you may see:

- **Item slot** for fuel or catalyst items.

- **Fluid gauge** with tooltip; **D** clears the tank when fluid is enabled.

- **Energy bar** when the coil draws stored FE.

- **Redstone mode** button (icon strip)—click to cycle behavior; hover reads the current mode.

Unused panels are hidden or masked so you only see what that coil actually uses.
